PHP DASAR

Selamat datang dihalaman php, disini forum untuk php, pembahasan dimulai dari basic, midle, sampai advance

Perulangan dengan while

 <?php
	$x = 1;
	while($x <= 5) {
		echo "The number is: $x <br>";
		$x++;
	}
?>

Perulangan dengan while dengan array

 <?php
	$array = ["data1", "data2", "data3", "data4", "data5", "data6"];	
	$x = 1;
	while($x < count($array)) {
	    
	    echo $array[$x];
	    $x++;
	}
?>

Perulangan dengan for

<?php 
	$array = ["data1", "data2", "data3", "data4", "data5", "data6"];	
	//perulangan dengan for
	for ($i=0; $i < count($array) ; $i++) { 
		echo $array[$i]."<br>";
	}
 ?>

Perulangan dengan foreach

<?php 
	$array = ["data1", "data2", "data3", "data4", "data5", "data6"];	
	//perulangan dengan menggunakan foreach
	foreach ($array as $key => $value) {
		echo $value;
	}
 ?>
